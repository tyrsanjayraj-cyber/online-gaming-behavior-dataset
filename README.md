 Gaming Analytics Project

Project Title

**Gaming Player Behaviour and Purchase Trend Analysis Using Python**

Project Description

This project analyzes player behavior in an online gaming environment using a gaming dataset. The objective is to understand how players interact with the game, analyze their playing patterns, and identify trends in in-game purchases and player performance.

By applying data analysis techniques using Python, the project extracts useful insights that can help gaming companies improve player engagement and optimize game strategies.


Objectives

* Analyze **player retention patterns**
* Study **in-game purchase trends**
* Identify **top players through leaderboard analytics**
* Visualize gaming behavior using data visualization techniques


Tools & Technologies Used

* Python
* Jupyter Notebook
* Pandas
* Matplotlib
* Seaborn
* GitHub

---

Dataset

Dataset used for this project:
Online Gaming Behavior Dataset

Dataset Source:
Kaggle – https://www.kaggle.com/datasets

The dataset contains information about:

* PlayerID
* PlayerLevel
* SessionsPerWeek
* PlayTimeHours
* InGamePurchases
* AchievementsUnlocked



Analysis Performed

Player Retention Analysis

Player retention was analyzed by examining the number of gaming sessions played by each player. This helps understand how frequently players return to play the game.

In-Game Purchase Trends

Purchase behavior was analyzed to understand how players spend money on in-game items and how purchase activity varies among players.

Leaderboard Analytics

Top players were identified based on their total playtime and gaming performance to create a leaderboard ranking.


Visualizations

The following visualizations were created:

* Player Retention Distribution
* In-Game Purchases by Player Level
* Top 10 Players Leaderboard

These visualizations help in understanding player engagement and game performance patterns.


Key Insights

* Players with higher session frequency show stronger engagement.
* Higher level players tend to make more in-game purchases.
* A small group of players contributes significantly to total gameplay time.
